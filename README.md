# UCL-Latex-Generator

## To run on local server :

- Git clone on local repo

From parent folder, run : 
- python -m venv .venv
- source .venv/bin/activate
- pip install -r requirements.txt
- python manage.py runserver

- Navigate to http://127.0.0.1:8000/
